## Project Background

While Midjourney is popular, the C-end competition is intense. The team pivoted to the B2B market: providing customized image generation APIs for enterprises.

## Product Form

An API service based on Stable Diffusion, supporting private model training, style customization, and batch generation. Core functions: model fine-tuning, API integration, and enterprise-level support.

## Implementation Process

**February 2024**: Technical validation  
**May 2024**: API launched  
**September 2024**: Signed 20 enterprise clients  
**December 2024**: Monthly revenue reached ¥120,000

## Monetization Model

API Usage Billing:
- Standard Model: ¥0.05/image
- Private Model: ¥0.2/image
- Enterprise Edition: Monthly subscription

Average Order Value: ¥12,000/month (B2B clients).

## Key Data

| Metric             | Value      |
| ------------------ | ---------- |
| Enterprise Clients | 25         |
| Monthly Revenue    | ¥120,000   |
| Avg Order Value    | ¥12,000/mo |
| Monthly API Calls  | 2 million  |

## Core Insights

1. **Avoid C-end Red Ocean**: B2B competition is lower with higher order values.
2. **Technical Barriers**: Private model training is the core advantage.
3. **Service as a Product**: Technical support is also a value point.
